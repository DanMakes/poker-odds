### Compiles and hot-reloads for development
```
/* 
how to fill players data manually
desktop: index 0-9
mobile: index 5-9 
*Note
In mobile, calculate data from index 5 to 9(players[5]~players[9]), so have to fill null data from index 0 to 4(players[0]~players[4])
*/

npm run serve
```
